## Overview

This is a Vulkan Hybird PBR ray-raster. To be more specific, a deffered PBR renderer + raytraced shadows.


|PBR (shadow off)| Shadow|
|:-----:|:-----:|
|![](img/final_result_PBR.gif)|![](img/final_result_shadow.gif)|


### Key features:

- Deferred rendering
- PBR
- Raytraced shadows for
    - point light
    - IBL diffuse light
    - IBL specular light
